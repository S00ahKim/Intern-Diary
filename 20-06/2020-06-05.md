### nginx
1. 설정 파일 위치
- 컴파일을 통해서 설치한 경우라면 기본적으로 /usr/local/nginx/conf 디렉토리에 위치
- apt-get을 이용해서 우분투에 설치한 경우는 /etc/nginx에 위치
- 설치 방법에 따라서 환경설정 파일의 위치는 다를 수 있기 때문에 아래의 명령을 이용하면 설정 파일의 위치를 쉽게 찾을 수 있다. 
```
sudo find / -name nginx.conf
```

2. 엔진엑스 재시작
restart 커맨드도 있지만, reload를 권장한다고 한다. reload를 진행하면 syntax 체크를 진행하고 성공하게 되면 older worker 프로세스를 중지 시키고 새로운 worker 프로세스를 만들어서 동작 시킨다고 하는데, 아마 그게 이유일 것 같다.


### filter_queryset
- queryset에서 모델 연결 시켜주고 그 모델에 대한 제어를 filter_queryset여기에서 한다고 이해하면 된다. 즉, queryset는 데이터 자체이고, filter_queryset는 데이터 가공
