### Function Annotation
1. python 3
2. 사용예
```
def func(arg1: str, arg2: 1+2, arg3: 'this is annotation') -> bool
```
- arg1 : 매개변수의 type
- arg2 : 간단한 연산
- arg3 : 문자열
- -> 리턴값의 형태 정의
3. 특징
- 강제성은 없다
- 주석일 뿐이며 해당 code 자체에는 어떠한 영향도 미치지 않는다. 
- 장점: 매개변수와 반환값을 명시적으로 볼 수 있다 (IDE 인식 가능)
