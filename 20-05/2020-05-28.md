### string -> datetime
1. 형식에 맞추어 포매팅하기
```
formatted_date = datetime.strptime(request_date, '%Y-%m-%d %H:%M:%S')
```
2. 타임존 맞춰 주기
```
formatted_date = formatted_date.replace(tzinfo=timezone.utc)
```
- 시간대를 찾을 수 있는 정보가 없는 데이트타임 객체를 naive 데이트타임이라고 하고, 시간대를 포함하면 aware 데이트타임이라고 함.


### 로드밸런싱
- 로드밸런스를 도입하기 위해서 자료 조사를 했다. 
- 현재 상황: 지금은 서비스 초기라서 문제가 발생할 경우 스케일 업만 하고 있다. 그래서 엔진엑스가 간단한 로드밸런서 역할도 하고 있다. 
- 진행 상황: 서버를 여러 대로 늘릴 예정이다. 그럴 경우 AWS에서 제공하는 로드밸런서를 쓸 수 있는데, 전통적인 방식도 있고 애플리케이션 단에서 로드밸런싱을 해주게 새로 나온 것도 있다. 둘이 방식이 조금 다르다. 전통적인 방식은 구성이 쉬운데 여러개를 둬야 할지도 모르겠고, ALB는 설정할 건 좀 있는데 하나로 가능하다고 한다.
