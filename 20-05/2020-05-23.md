### 형태소 분석기 비교
1. mecab
- 사용자 사전을 정의하고 중요도 설정 가능
- 지속적으로 사전이 업데이트됨
- 사용자 사전을 구축하고 토크나이징 되는 것을 확인하면서 중요도를 갱신시키는 과정이 필요함
- 연산 속도가 좋음
- 분석 품질 상위권
- "은전한닢" 오픈소스 프로젝트 (윈도우 환경일 경우 작동이 어려움 -> 코랩에서 돌리면 됨)

2. 꼬꼬마
- 사용자 사전을 정의할 수 있음
- 시간이 너무 오래 걸림
- 분석 품질 상위권 (정확한 품사 정보)

3. 카이
- 사용자 사전을 정의하기 어려움
- 카카오에서 만든 형태소 분석기
- 띄어쓰기가 없는 문장의 경우 성능이 좋지 않음

4. koNLPy는 꼬꼬마, 코모란, 트위터, 한나눔, 은전한닢을 묶어서 편하게 사용할 수 있도록 함.
