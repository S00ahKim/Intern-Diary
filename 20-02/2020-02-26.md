* Django AssertionError
1. if-else 문으로 리턴 처리를 다르게 할 때 등의 상황에서 한쪽 분기에서 return할 것이 없을 경우 발생함.


* 프론트에 전달하는 데이터 타입
1. 일반적으로 프론트와 상의해서 전달함
2. 다른 곳에서 이 데이터를 사용하고 있다면 해당 데이터 타입을 사용하는 것이 바람직함. 최대한 다른 곳에서 쓰는 데이터 타입과 맞춰서 보내는 것이 좋음.
3. 날짜나 시간 같은 특정한 데이터 타입을 디폴트 값으로 줄 경우 합의된 데이터를 전달한다. (ex.1970년 1월 1일)
