### Django 쿼리셋 결과 합치기
1. union()
```
result = a.union(b, all=True)
```
- 두 번째 인자는 중복을 허용할지에 대한 것 (디폴트: False)
- 리턴 타입이 쿼리셋 리스트 타입이기 때문에 ORM 형식을 그대로 사용할 수 있다.

2. | 연산자
```
result = a | b
```
- 리턴 타입이 쿼리셋 리스트 타입이기 때문에 ORM 형식을 그대로 사용할 수 있다.

3. + 연산자
```
a = list(a)
b = list(b)
result = a + b
```
- 각 쿼리셋 리스트 타입을 일반 리스트 타입으로 변경한 후 합침
